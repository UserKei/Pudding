# Game Assets

This directory contains cleaned, project-ready assets referenced by Godot scenes.

Suggested categories:

```text
assets/game/player/
assets/game/props/
assets/game/tilesets/
assets/game/backgrounds/
assets/game/audio/
assets/game/ui/
```

Do not put unreviewed downloaded packs directly here. Keep raw packs in `assets/third_party/` first.
